# Gigaflop PP3

Este proyecto es una plataforma de gestión de clientes y cotizaciones. 

##  Instalación

Clonar el repositorio y ejecutar los siguientes comandos en `client/` y `server/`:

```sh
git clone https://github.com/
npm install
cd ../server
npm install

***********************************************************************************

Para correr servidor: 
cd server
npm run dev

Para correr cliente:
cd client
npm start
